## 프로젝트 개요

RealSense D455 카메라로 입력된 실시간 영상에서 객체를 검출하는데 추론 시 한정되지 않은 어휘로 새로운 클래스를 감지하여 바운딩 박스와 함께 텍스트를 출력하고 세크멘테이션 마스크 영역을 시각적으로 오버레이하는 프로그램을 작성한다. 
최신 Yolo26 버전을 통해 실시간 처리가 가능하도록 한다.

## 개발 환경

| 항목 | 사양 |
|------|------|
| **OS** | Windows 11 |
| **GPU** | NVIDIA RTX 5070 |
| **카메라** | Intel RealSense D455 |
| **Python** | 3.10 이상 권장 |
| **CUDA** | 12.x (RTX 5070 Blackwell 아키텍처 호환 버전) |
| **cuDNN** | CUDA 버전에 맞는 최신 버전 |


## 실행 방법

```bash
# 1. 가상환경 생성 (권장)
python -m venv venv
venv\Scripts\activate

# 2. 패키지 설치
pip install -r requirements.txt

# 3. 실행
python main.py
```

---

## 예상 출력 화면

- 카메라 영상 위에 검출된 객체마다 반투명 컬러 마스크가 씌워짐
- 각 객체의 바운딩 박스와 상단에 `클래스명 confidence` 텍스트 표시
- 좌측 상단에 FPS 카운터 표시
- `q`로 종료, `s`로 스크린샷 저장

## 확장 가능 기능 (선택사항)

1. **뎁스 정보 활용**: 검출된 객체까지의 거리(m)를 텍스트에 함께 표시
2. **객체 추적**: ByteTrack 등을 연동하여 동일 객체에 고유 ID 부여
3. **녹화 기능**: `r` 키로 결과 영상을 MP4로 녹화
4. **웹 스트리밍**: Flask/FastAPI로 결과 영상을 웹 브라우저에 스트리밍
5. **커스텀 모델**: 특정 도메인 객체 검출을 위한 fine-tuned 모델 적용
